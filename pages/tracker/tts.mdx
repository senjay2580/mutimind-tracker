# 语音合成 (TTS)

| 项目名称 | GitHub 地址 | Stars | 描述 |
|---------|------------|-------|------|
| GPT-SoVITS | https://github.com/RVC-Boss/GPT-SoVITS | 56,391 | 1 分钟语音数据即可训练高质量 TTS（少样本克隆） |
| ChatTTS | https://github.com/2noise/ChatTTS | 39,013 | 面向日常对话的生成式语音模型 |
| fish-speech | https://github.com/fishaudio/fish-speech | 29,082 | SOTA 开源 TTS 引擎 |
| VoxCPM2 | https://github.com/OpenBMB/VoxCPM | 17,199 | **Tokenizer-Free TTS**（OpenBMB / 面壁，Apache 2.0 商用友好）。2B 参数，200 万小时多语种语音训练，**30 种语言 + 9 种中文方言**（四川话/粤语/吴语/东北话/河南话/陕西话/山东话/天津话/闽南话）。三种使用方式：① **Voice Design** 自然语言描述（性别/年龄/语调/情绪/节奏）凭空造新声音，无需参考音频；② **Controllable Cloning** 短参考音频克隆 + 可选风格引导；③ **Ultimate Cloning** 给参考音频 + 转录，无缝续写保留全部细节。**48kHz 录音棚级音质**，支持 16kHz 输入超分到 48kHz。RTX 4090 上 RTF ~0.3，配 [Nano-vLLM](https://github.com/a710128/nanovllm-voxcpm) 或 vLLM-Omni 加速到 ~0.13。基于 MiniCPM-4 backbone，端到端 diffusion autoregressive 直接生成连续语音表征（不走离散 token）|
