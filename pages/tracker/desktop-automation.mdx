**Computer-Use Agent 基础设施** — 给 AI 代理提供沙箱、SDK、桌面自动化和 benchmark，让代理能看屏幕、点按钮、打字并执行完整 GUI 工作流。拆成 Cua Driver、跨平台 sandbox API、CuaBot、Cua-Bench、Lume 等模块，覆盖 macOS / Linux / Windows 的桌面控制与评测。偏 agent infra / runtime / benchmark 平台，不是装了就提效的普通插件。

**注意**：⚠️ 偏底层基础设施与实验平台，上手成本高于普通 Agent 应用；macOS 相关能力依赖 Apple Silicon / virtualization 生态。

**桌面级 Computer-Use Agent** — 对标 Anthropic Computer Use + OpenAI Operator，让 LLM 通过截屏 + 鼠标键盘控制操作真实桌面（Windows / macOS / Linux）。GUI Agent + 浏览器 use + 桌面 use 三合一，MCP 兼容可被 Claude Code / Cursor 调用。典型用例：自动跑桌面办公（填表 / 截屏 / 操作 SaaS）、不可 API 化软件的自动化。TurixAI 团队出品。

**基于视觉语言模型的 Android 自动化 Agent** — 手机端 Computer-Use Agent：VLM 看截屏 + 推导坐标 + ADB 模拟点击，让 LLM 自主操作真实 Android 手机。不依赖 accessibility service（避开应用反爬限制），用纯视觉 + adb shell 通用接口。典型场景：手机自动化测试、抢券脚本、社交 App 互动机器人、跨 App 工作流。

**注意**：⚠️ 2026-01 后未更新；仅 Android。
