**Simon J.D. Prince 的《Understanding Deep Learning》教科书配套仓库** — MIT Press 2023 出版，**免费 PDF 永久公开**（[udlbook.github.io](https://udlbook.github.io/udlbook/)）。~600 页，41 章，是**现代深度学习教科书的顶流**：相比 Goodfellow 2016 "花书" 更新覆盖 **Transformer / Diffusion / GAN / RL / Graph NN / Fairness / 因果推断**等 2020 后核心内容，相比 Bishop PRML 更偏工程实践。

**仓库内容**：每章配套的 **Jupyter notebook 习题 + 答案 + 实验代码**，直接 Colab 跑。

**和谁同档**：
- **Goodfellow 花书**（2016，理论扎实但旧，缺 Transformer）
- **Bishop PRML / PRDL**（2006/2024，偏数学概率）
- **d2l.ai 动手学深度学习**（中文社区首选，更偏实操但章节略浅）
- **UDL** ⭐ **2023 现代覆盖最完整 + PDF 免费 + 配套 notebook 齐**

**适合**：
- 想系统学一遍现代深度学习（不是只追 transformer 论文）
- 教学 / 自学路径主干教科书
- 面试准备 / 研究入门基础

**注意**：⚠️ NOASSERTION（书 PDF 是 MIT Press 版权 + 作者特许免费 PDF，notebook 可改用），二次分发书本身要看 MIT Press 授权；notebook 引用作者即可。
