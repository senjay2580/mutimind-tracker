**亮点**：
- 原生 `.fig` 文件读写，支持组件/变体/实例，可在 Figma 和 OpenPencil 之间拷贝粘贴节点
- AI 驱动：接入 OpenRouter / Anthropic / OpenAI / Google AI 等多 Provider，聊天即设计
- 可编程：headless CLI + MCP Server + Figma Plugin API `eval`，适合做 design-to-code、Lint、Token 提取等自动化流水线
- Tauri + Skia + WebGPU 渲染，跨平台桌面应用 + Web 版同步提供

**注意**：⚠️ 项目仍处于活跃开发中（Active development），尚未进入生产可用阶段。
