## [AutoFigure-Edit](https://github.com/ResearAI/AutoFigure-Edit)

<RepoMeta
  url="https://github.com/ResearAI/AutoFigure-Edit"
  stars={3297}
  updated="2026-05-17"
  lang="Python"
  license="MIT"
  tags={['scientific-figures', 'svg', 'paper-writing', 'research']}
/>

**一句话定位** — 从论文方法文本生成可编辑科研插图的工具，把 method section 转成 editable SVG figures，并支持后续编辑。它面向科研写作和论文图表生产，可用于把实验方法、模型结构、流程图快速转成可修改的矢量插图。

**亮点**：
- 目标不是静态图片，而是可编辑 SVG，便于论文排版和二次修改
- 配套 AutoFigure / FigureBench 方向，适合科研图生成评测与自动化写作流程
- 对需要批量产出模型结构图、实验流程图的研究 Agent 很有价值

**注意**：⚠️ 科研插图需要事实校验；自动生成图不能替代作者对方法细节和视觉表达的最终审阅。

**一句话定位** — 面向 arXiv 论文搜索与分析的 MCP Server，让 Claude、GPT、各类 Agent 通过标准 Model Context Protocol 查询论文、读取元数据并辅助研究。适合把 arXiv 作为 Agent 的实时论文检索工具接入文献综述、选题和论文阅读流程。

**亮点**：
- 以 MCP server 形式提供，方便接入 Claude Desktop、Claude Code 或其他 MCP 客户端
- 聚焦 arXiv 搜索、论文信息获取和研究辅助分析
- 适合搭配文献综述 Agent、研究日报、论文推荐和自动摘要工具
