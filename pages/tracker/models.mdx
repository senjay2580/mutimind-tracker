**AI powered speech denoising and enhancement** — 商业 TTS 公司 Resemble.ai 开源的**双段管线**：**Denoiser**（去环境噪声 / 室内回响 / 嘶嘶声）+ **Enhancer**（44kHz 重采样 + 频带补全 + 韵律修复），一次性把糟粕录音抬到录音棚级。**关键差异点**：同类工具（DeepFilterNet / RNNoise）只做 denoise，Resemble Enhance **denoise + super-resolution 一体**——专治"手机随手录 / 远场麦 / 老视频里抠人声"这种又脏又糊的料。pip install 即用，给 8s 片段或长 wav 都行。**做 TTS 训练数据清洗 / 播客后期 / 老素材修复** 直接命中。

**Versatile audio super resolution (any → 48kHz)** — 作者 Haohe Liu（同款 VoiceFixer 作者），**唯一一个"通用"音频超分**：不分人声 / 音乐 / 环境音 / 音效，任意输入采样率（2k / 4k / 8k / 16k / 24k / 32k）**统一抬到 48kHz**。基于 latent diffusion，模型只有 100MB 左右，**比传统 BWE / NU-Wave 跨域泛化好得多**。典型场景：**老电台 / 老电影低码率音频抢救、AI 生成低质音频后处理、压缩失真复原、做 hi-fi 数据集**。和 Resemble Enhance 区别：那个偏人声 denoise + enhance 一体，AudioSR 是**纯频带恢复**，可叠加使用。
