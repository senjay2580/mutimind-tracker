| 名称 | 网址 | 用途 |
|------|------|------|
| **Reddit** | https://www.reddit.com/ | **真实用户痛点 / 抱怨 / 需求的金矿**。**搜某产品 + "issue / problem / hate / broken / alternative"** 直接挖到一手用户原声 — 比看官方文档诚实 100×。子版块（r/Entrepreneur / r/SaaS / r/SideProject / r/StartUps）是产品想法的早期信号源，**找 PMF / 选品 / 反向求购**必到 |
| **Google Trends** | https://trends.google.com/ | **关键词热度 + 地域分布**官方数据。**判断"我这个 idea 是上升趋势还是衰退"** 的客观依据。可对比多关键词时序曲线 + 看不同国家 / 城市搜索强度，**适合选品 / 内容选题 / 命名 / 投放区域**决策。免费、无 key、数据延迟 1-3 天 |
| **TikTok Creative Center** | https://ads.tiktok.com/business/creativecenter/ | **TikTok 官方趋势平台**（广告主入口但公开访问）。**Top Ads / 热门音乐 / 热门 hashtag / 创意洞察 / 商品趋势** 全方位看 TikTok 当下吃香的内容形态。**短视频 / 跨境电商 / 出海团队** 选题必看。比抖音商业版（巨量算数）更直接 |
| **Product Hunt** | https://www.producthunt.com/ | **每日新产品 / 工具发布榜单**（硅谷创业风向标）。**看每天的 Top 5 + Newsletter** 就能跟上整个独立开发者圈在做什么。**适合**：找新工具替代旧工作流 / 看竞品发布日 / 学习 launch 文案 / 给自家产品找 launch 时机。**Show HN + PH 双登陆**是独立开发者标配 |
| **GitHub Trending** | https://github.com/trending | **每日 / 每周 GitHub 最热仓库**（按语言 / 时间窗口筛）。**技术早期信号**——开发者真正花时间关注的项目（不是被刷上 HN 头条的）。**结合 awesome-list / Hacker News / Twitter dev 圈** 三路交叉验证。⚠️ 国内访问可能需代理 |
| **Hacker News** | https://news.ycombinator.com/ | **硅谷工程师 / 创业圈讨论中心**（YC 旗下）。**评论质量行业最高** — 一线大厂工程师 / 创始人 / VC 直接对线，常有"被讨论的产品的作者亲自下场回复"。**Show HN / Ask HN / Front Page** 三栏分别用于发布产品 / 求问题 / 看热度。**全球技术 + 创业的事实公告板** |
| **Substack** | https://substack.com/ | **独立作者长文订阅平台**（newsletter 复兴的代表）。**深度观点的源头** — Stratechery / Lenny's Newsletter / Not Boring / 各类硅谷分析师 / VC 个人媒体都在 Substack。**适合订阅 5-10 个领域专家**长期跟踪，建立深度洞察。**和 Twitter 区别**：Substack 给思考深度更长的内容空间 |
| **Meta Ad Library** | https://www.facebook.com/ads/library/ | **Meta 官方广告透明库**（公开免费）。**直接搜索竞品在 Facebook / Instagram 投的所有广告素材**（图片 / 视频 / 文案 / 投放时间），**完全合规零灰色**——这是 Meta 强制广告主透明的法定数据。**电商 / DTC 品牌 / 跨境** 选品和找爆款素材的杀手锏 |
| **AnswerThePublic** | https://answerthepublic.com/ | **搜索引擎自动补全的可视化聚合**。**输入一个关键词** → 自动呈现"who / what / when / where / why / how + 关键词" 等所有真实用户搜索补全 → 画成可视化树状图。**适合选题 / SEO / 内容营销** — 找出用户**实际**在问什么问题。免费档每天 3 次搜索，付费 $9+/月 |
| **Pinterest Predicts** | https://business.pinterest.com/pinterest-predicts/ | **Pinterest 官方年度趋势预测报告**。**视觉 / 生活方式趋势先行指标** — Pinterest 上的 Pin 行为通常领先现实趋势 6-12 个月。**家居 / 时尚 / 美妆 / 婚礼 / 美食 / 育儿** 等垂类品牌、电商和广告创意必看。免费阅读，每年发布 |
