### 网盘 & 下载工具

## [netdisk-fast-download (NFD)](https://github.com/qaiu/netdisk-fast-download)

**后端直链解析服务** — Java/Vert.x 开发，需要服务器部署。把网盘分享链接转为可直接下载的直链（302 跳转或 JSON）。支持 20+ 网盘（蓝奏云、123盘、夸克、Google Drive、OneDrive 等）。适合站长/开发者提供公共解析 API。

## [OpenList](https://github.com/OpenListTeam/OpenList)

**多网盘聚合文件管理平台** — AList 的社区分支（因治理问题独立），Go + SolidJS 开发。支持 50+ 种存储（阿里云盘、百度、S3、WebDAV 等），提供统一 Web 界面进行文件浏览、预览、上传、下载、WebDAV 挂载。功能最全面，适合搭建私有网盘门户。

## [LinkSwift](https://github.com/hmjz100/LinkSwift)

**浏览器油猴脚本** — 零部署，装 Tampermonkey 即可。在网盘官网页面上直接提取直链，配合 IDM/Aria2 加速下载。支持 8 个主流国内网盘（百度、阿里、夸克、123盘等）。适合普通用户日常使用。

## [zmap](https://github.com/zmap/zmap)

<RepoMeta
  url="https://github.com/zmap/zmap"
  stars={6208}
  updated="2026-05-21"
  lang="C"
  license="Apache-2.0"
  tags={['internet-scanning', 'port-scanner', 'security-research', 'stateless']}
/>

**互联网级单包端口扫描器** — 密歇根大学 Halderman 团队出品，2013 USENIX Security 论文级里程碑工具。**stateless single-packet 设计**：千兆网卡 **45 分钟扫完整个 IPv4 公网** 一个端口，10G + netmap 仅 5 分钟。支持 TCP SYN / ICMP / DNS / UPnP / BACNET / UDP 探针。**和 nmap 方向不同**：nmap 深扫单主机，ZMap 浅扫全互联网；要拿 banner / TLS 证书必须配套 [ZGrab2](https://github.com/zmap/zgrab2)。

**配套生态**：Censys.io 商业全网测量服务 + 国内外安全研究 / 红队侦察 / 学术测量必备。

**亮点**：
- 全网级扫描速度，把"互联网测量"从理论变日常
- 用户态网络栈（netmap / PF_RING）绕过内核
- 多探针模块化，可写自定义 UDP payload

**注意**：⚠️ 未经授权扫别人 IP = 违法（《网络安全法》/ CFAA / GDPR）；公网扫描会被运营商和 CDN 风控立即记下；要 root；Windows 原生不支持，需 Linux/macOS/BSD/WSL2。

**Fork**：[senjay2580/zmap](https://github.com/senjay2580/zmap)（已 fork 备份）

## [zgrab2](https://github.com/zmap/zgrab2)

<RepoMeta
  url="https://github.com/zmap/zgrab2"
  stars={2117}
  updated="2026-05-20"
  lang="Go"
  license="No License"
  tags={['internet-scanning', 'banner-grab', 'tls-handshake', 'zmap-ecosystem']}
/>

**ZMap 姊妹项目：Fast Application Layer Scanner** — 用 Go 写的有状态应用层扫描器，**专门补 ZMap 缺的那一段**：ZMap 只发单包探测"端口是否开"，ZGrab2 接力做**完整 TCP 握手 + 协议级抓取**——HTTP banner / TLS 证书链 / SSH key / FTP welcome / SMTP greeting / MySQL handshake 等。**标准 pipeline**：`zmap -p 443 | zgrab2 tls`，先全网筛活的，再批量抓握手。

**核心模块**：http / tls / ssh / ftp / smtp / mysql / postgres / mongodb / redis / telnet / modbus（工控）/ bacnet（楼宇自控）。

**亮点**：
- ZMap 工作流的"第二段"，配套使用才是完整侦察链
- Go 写，跨平台编译方便（Linux/Mac/Windows 都能跑）
- 输出标准化 JSON，方便接 Elastic / ClickHouse 做大规模分析
- 是 Censys.io 全网证书库背后的工程实现

**注意**：⚠️ NOASSERTION（无标准 license，二次分发要看 LICENSE 文件原文）；扫公网仍然受法律约束，参考上方 zmap 同款警告。

**Fork**：[senjay2580/zgrab2](https://github.com/senjay2580/zgrab2)（已 fork 备份）
