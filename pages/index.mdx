## 添加新项目

在 Claude Code 中调用 `github-tracker` skill：

```
/github-tracker add https://github.com/<owner>/<repo>
```

skill 会自动：
1. 用 `gh api` 拉取 stars / 描述 / 主语言 / license
2. 判断分类（基于 README 内容 + 已有分类语义）
3. 写入对应 `pages/tracker/<category>.mdx`，使用统一的 `<RepoMeta>` 卡片格式
4. `git commit` + `git push` → Vercel 自动构建上线

## 维护原则

- **唯一数据源**：本仓库的 `pages/tracker/*.mdx`，原 `MutiMind/Input/*.md` 已归档为只读
- **格式统一**：每个新条目用 `## [name](url)` 标题 + `<RepoMeta>` 元信息卡片 + 描述段落
- **历史保留**：旧条目保持表格格式，新条目走卡片格式，混排不影响阅读
